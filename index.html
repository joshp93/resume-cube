What Think Ye?

I designed this app to get some feedback when giving presentations at my church. It is a similar concept to Slido, but it visualises responses differently and stresses the fact that it doesn't collect any data about the user other than their response.
This is written in Angular and uses Angular Material for the forms, buttons and inputs. It uses Google's Firebase platform for Authentication and Database storage. On the visualisation page, data is streamed in realtime from the Firestore database, so new responses can be seen as they come in.
I can provide a demo of this application on request.

CSN Second Contacts

I wrote this CRM system on request for CSN, a small charity, because I wanted to support their efforts. They are now using this system fulltime and it is saving them hundreds of pounds a month. It is a database of Second Contacts information which can be used for referrals.
The CRM system is written in Angular with Angular Material components. I designed a custom pallete and colour theme for them based on their logo. It uses Google's Firebase for Authentication and Database storage.
